# Aelaki Dictionary (Discord Sources)

All entries are sourced from Discord messages by kamidere in the Conlangs Discord Network (2024-07 to 2026-01). Dates indicate the message where the form was first attested. Entries are organized by lemma (base/citation form) where identifiable; inflected forms are listed under their lemma.

**Notation:**
- **Bold** = lemma / citation form
- `SMALL CAPS` gloss = grammatical abbreviation
- (date) = Discord message date
- Root notation: C1-C2-C3 or C1-C2-C3-C4

**Stative prefix system:**
- Ho- = probable / about to (doubled: Hoho- = keeps being about to)
- Nü- = inchoative / begin (doubled: Nünü- = in process of beginning)
- Nï- + umlaut = cessative / stop (doubled: Nïnï- = in process of stopping)
- Lu- = resumptive / resume/remember (doubled: Lulu- = in process of resuming)
- Ko- = almost / nearly (doubled: Koko- = keeps almost doing)

**Stative conjugation:** Agreement suffix at end (not prefix like active verbs). -oth = 1p.FEM.SING. Evidentials: -ro (past auditory), -mu (future inferential), -ya (optative future), -shë (past visual) (2025-06-04).

## Grammatical Notes Summary

### Word Order (2025-05-30, 2025-06-05)
- SVO for transitive verbs
- SV for active intransitive
- VS for stative intransitive
- SOV was original proto-Aelaki order (preserved in imperative morphology)

### Morphological Strategies
| Operation | Noun meaning | Verb meaning | Adjective/Adverb meaning |
|-----------|-------------|-------------|------------------------|
| Reduplication (C2) | Plural | Habitual (transitive) / Continuous (stative) | Comparative |
| Umlaut (whole word) | Collective | Perfect (transitive) / Gnomic (stative) | Superlative |
| f-infix | Zero number | Negative/Subjunctive | Negative |
| Unmarked | Singular | Base form | Positive |

### Alignment
Active-stative alignment. Active verbs prefix subject agreement; stative verbs suffix it. Inanimate nouns cannot be subjects of transitive or active verbs (2025-06-04).

### Name of the Language
"Aelaki is really pronounced Alaku" by non-speakers — the name itself is specifically in a collective plural form (2025-06-09).
